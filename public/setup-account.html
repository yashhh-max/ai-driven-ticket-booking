🎬 Account System Setup

Choose your setup method and follow the steps below

⚡ Method 1: Automated Setup (Recommended)

The fastest way to set up your database. Requires Node.js installed.

npm run setup

Or use the interactive wizard:

npm run setup:wizard
⚠️ Prerequisites: You need to add your Supabase Service Role Key to .env.local:
SUPABASE_SERVICE_ROLE_KEY=your_key_here
Get it from Supabase Dashboard → Project Settings → API Keys → Service Role (secret)

🔧 Method 2: Interactive Wizard

Step-by-step guided setup with helpful prompts and fallbacks.

npm run setup:wizard

This will guide you through:

  • Database setup (automatic or manual)
  • Header navigation integration
  • Feature testing checklist

📋 Method 3: Manual Dashboard Setup

For maximum control, set up directly in Supabase Dashboard.

  1. Go to Supabase Dashboard
  2. Open your project
  3. Click SQL Editor
  4. Click New Query
  5. Copy-paste the SQL (see below)
  6. Click Run
📝 SQL Script: Use the SQL from scripts/020_create_account_system.sql

✅ Verify Your Setup

Run this query in Supabase SQL Editor to verify all tables were created:

SELECT table_name FROM information_schema.tables 
WHERE table_schema = 'public' 
AND (table_name LIKE 'user_%' 
  OR table_name = 'promo_codes' 
  OR table_name = 'loyalty_points_history' 
  OR table_name = 'points_redemption' 
  OR table_name = 'password_change_history')
ORDER BY table_name;

You should see all 12 tables:

  • loyalty_points_history
  • password_change_history
  • points_redemption
  • promo_codes
  • user_loyalty_points
  • user_payment_methods
  • user_profiles
  • user_promo_usage
  • user_refunds
  • user_saved_locations
  • user_transactions
  • user_wishlist
⏱️ Time to verify: Run the query above in Supabase Dashboard. If you see all 12 tables, you're good to go! ✓

🚀 Next Steps

2️⃣ Test Your Account System

Navigate to: http://localhost:3000/account

Test these 8 features:

  • Profile - Edit name, phone, bio, picture, date of birth
  • Saved Locations - Add home, work, or theatre addresses
  • Wishlist - Add/remove movies
  • Payment Methods - Save credit/debit cards
  • Transactions - View booking history
  • Offers & Promo Codes - Browse discount codes
  • Loyalty Points - View balance and redemption
  • Settings - Change password or delete account
